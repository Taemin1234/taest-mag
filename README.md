# 테이스트 매거진


## 소개

Taest Magazine은 다양한 분야의 에디터가 글을 게시하고 독자가 카테고리별로 탐색할 수 있는 온라인 매거진 플랫폼입니다.
백엔드와 프론트엔드를 모두 직접 구현하며 콘텐츠 기반 웹서비스의 구조를 이해하는 것을 목표로 했습니다.


## 기술스택

- 프론트엔드 : Next.js (15v), react(19v), CSS Modules, Swiper
- 백엔드 : Node.js + Express
- DB : MongoDB + Mongoose

- 배포 : Vercel, AWS EB

- 이미지 저장 : Cloudinary
- 글 작성 에디터 : tiny MCE
- 기타툴 : JWT


## 주요 기능

- JWT 기반 로그인 / 회원가입
- 에디터별 게시글 목록 제공
- 카테고리 필터링 및 검색 기능
- 이미지 업로드 (Cloudinary)
- 반응형 UI & Swiper 슬라이더
